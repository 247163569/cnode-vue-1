# cnode-vue

> Vue版 cnodejs.org社区 WebApp [线上访问](https://hanyang.me/cnode)

## Build Setup

``` bash
# clone repo
git clone git@github.com:ihanyang/cnode-vue.git

# install dependencies
npm install

# build for production
npm run build
```
